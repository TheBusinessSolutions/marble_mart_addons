POS Smart Cash Connector

Connect Odoo to a smart draw

Main features:
  • Define a new payment method called smartcash
  • Using the SmartCash payment method allows you to connect to the different smart drawers on the market
  • To connect to the drawer, this module connects to the gateway.
  • You dont't need IOTBOX
  • You can cancel the payment from the POS of Odoo.

How it works

When installing the module, You must create a payment method called SMARTCASH.

The connector is designed with a new architecture based on the POS payment terminals. Therefore the communication is made from the web browser.

This payment method must be configured manually in the POS where you want to use .

The module establishes the connection between Odoo and the gateway, which is the link between Odoo and the drawer. The check allows you to go directly to the ticket once paid. The installation of the gateway license and configuration is included in the module price, and it will be configurated by our technical team.

Once the gateway is configured, SMARTCASH can be used as a payment method to charge its customers.

Pressing the SMARTCASH payment method passes control to the gateway and therefore to the drawer asynchronously, so that we send the message and we can continue working with the POS.

You can cancel the payment in any moment you want from Odoo POS.

You can retry the payment after cancelling it.

At the time the payment is made in the drawer, we will be notified immediately that the payment is made and we can validate it.
