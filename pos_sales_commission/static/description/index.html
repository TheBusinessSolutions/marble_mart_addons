POS Sales Commission Odoo App

Sales Commission for Point of Sales POS Orders

This app provides features to create and manage POS sales commission for POS orders.

Main Features:
  • This app allows your company to manage POS sales commissions with different options as shown below screenshots.
  • You can configure after installing the module which option your company is following to give POS commissions to your POS User and POS Manager while making POS Orders.
  • We are allowing you to select When to Pay and Calculation based so you can configure a policy that will be used to give commission to the POS team.
    • For every Calculation based on we are allowing you to select:
      • Sales Manager Commission %
      • Sales Person Commission %
  • If you keep the Sales Manager Commission 0% then the system will not create sales commission lines and sales commission worksheets. The system will work only with a single option at a time i.e., for example, you can select When to Pay = POS Order Paid and calculation based on = POS product category so the system will run on that and create commission entries.
    • POS Commission workflow will be:
      • Option1 :
        • POS Order Paid -> Create Sales commission worksheet (if not created for the current month) -> Add Sales commission lines on worksheet -> For every POS sale of the current month, the system will append sales commission lines on the worksheet. -> End of month Account may review worksheet and create commission invoice to pay/release commission to POS salesperson and POS sales manager/leader.
  • Commission to salesperson/leader always will be in company currency. This module supports multi-currency so POS orders can be in different currencies but the system will take care of it and create commission lines in company currency.
  • Sales Commission products are created using a data file in the module but you still can change commission products on a commission worksheet to create commission invoices.
  • Commission amount will be based on the Net Revenue Model which considers amounts without taxes. (This module does not support Gross Margin Model)
  • For more details please check below screenshots and watch the video.
