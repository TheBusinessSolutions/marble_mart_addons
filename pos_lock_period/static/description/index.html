Point of Sale POS Lock Period

Point of Sale Lock - Odoo App

This app allow you to lock point of sale for certain period/time. You can specify locking period by Time, Date, DateTime and Day.

Main Features:

  • Allow your POS manager to lock Point of Sales for certain period of time. And if user try to create new session or resume old session then system will not allow him/her to do that and raise warning message.
  • POS manager can activate or deactivate locking configuration.
  • POS manager can create multiple locking period configuration and all will be considered to check locking during new session and resume session.
  • Note: Timezone must be set on all POS users. So go to user form and set timezone.
  • Allow your POS manager to specify locking period in Time duration, Date duration, Datetime duration and Days.
